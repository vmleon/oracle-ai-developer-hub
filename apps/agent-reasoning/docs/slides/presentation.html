The Gap in LLM Reasoning

Large language models predict the next token. They don't plan, backtrack, or verify their own logic. When faced with multi-step problems, they confabulate. When asked to self-correct, they often make things worse. The raw generation loop wasn't built for structured thought.

"From predicting the next token to predicting the next thought."

Agent Reasoning wraps any Ollama-served model in cognitive architectures drawn from peer-reviewed research. The model stays the same; the thinking process around it changes.

I

Cognitive Architectures

16 reasoning strategies, each grounded in published research

Core Strategies

☰

Chain-of-Thought

Step-by-step prompt injection. Math, logic, explanations. Wei et al. 2022.

☆

Tree of Thoughts

BFS branch exploration with heuristic scoring and pruning. Yao et al. 2023.

⚙

ReAct

Interleaved reasoning + tool use. Web search, calculator, Wikipedia. Yao et al. 2022.

✎

Self-Reflection

Draft, critique, refine loop. Multi-turn improvement. Shinn et al. 2023.

⚒

Decomposed

Break complex queries into sub-tasks, solve sequentially. Khot et al. 2022.

☑

Self-Consistency

Generate k samples, majority vote. Temperature-sampled diversity. Wang et al. 2022.

Advanced Strategies

↺

Refinement Loop

Generator, Critic (0.0-1.0 score), Refiner. Iterates until quality threshold met.

◆

Complex Refinement

5-stage pipeline: Accuracy, Clarity, Depth, Examples, Polish. Production-grade.

⚔

Adversarial Debate

Pro/con debate rounds with LLM judge. Irving et al. 2018.

♜

MCTS

Monte Carlo Tree Search with UCB1. Complex strategy and planning. Browne et al. 2012.

★

Analogical

Structural analogy mapping. Cross-domain problem solving. Gentner 1983.

?

Socratic + Meta

Progressive questioning, plus auto-classification that routes to the optimal strategy.

The Interceptor Pattern

The ReasoningInterceptor acts as a drop-in replacement for any Ollama client. It parses a compound model string (like gemma3+cot), extracts the base model and strategy, then routes to the correct agent. No code changes needed in downstream apps.

# Strategy appended to model name with "+" response = client.generate( model="gemma3:270m+tot", prompt="Measure exactly 4 gallons..." ) # Interceptor splits: model=gemma3:270m, strategy=tot # Routes to ToTAgent automatically

Multi-Call Patterns

Simple agents make one LLM call. The interesting ones make many. Each strategy modifies the prompt, not the model, to steer cognition.

Single-Pass

  • Standard: Direct generation
  • CoT: Inject "think step-by-step"
  • Least-to-Most: Easy-to-hard sequencing

Multi-Pass

  • ToT: Generate → Evaluate → Prune
  • Reflection: Draft → Critique → Improve
  • Consistency: k samples → Vote
  • Refinement: Generate → Score → Refine
  • Debate: Pro → Con → Judge

Four Ways In

CLI, TUI, Python API, and Gateway Server

Interactive CLI

The primary interface. Rich-powered menu with all 16 agents, arena mode, head-to-head comparisons, benchmarks, and session history. Every response shows TTFT, total time, and tokens/sec.

# Full interactive menu agent-reasoning # Jump directly to specific modes agent-reasoning --arena # All agents race agent-reasoning --head-to-head # Compare two agent-reasoning --benchmark # Performance testing

Terminal UI

A Go-based TUI built with Bubble Tea and Lipgloss. 7 views, 10 structured visualizers, ~9,800 lines of Go. Auto-starts the reasoning server on launch.

Views

  • Chat (split-pane, live streaming)
  • Arena (4x4 grid, 16 agents racing)
  • Head-to-Head Duel + LLM Judge
  • Step-Through Debugger

Features

  • Structured visualization per agent type
  • Hyperparameter tuner (press p)
  • Strategy advisor (press ?)
  • Benchmark dashboard with 4 tabs

Python API & Gateway

Python Library

from agent_reasoning import ( ReasoningInterceptor ) client = ReasoningInterceptor() resp = client.generate( model="gemma3+cot", prompt="Explain quantum..." )

Gateway Server

FastAPI proxy impersonating Ollama. Any Ollama-compatible app (LangChain, Web UIs) gains reasoning without code changes.

agent-reasoning-server --port 8080 # Now point any app at: # http://localhost:8080 # model: gemma3:270m+tot

Key Findings

  • CoT achieved the highest average accuracy (87.0%), outperforming Standard on GSM8K (+6.6%) and MMLU (+6.7%)
  • Self-Consistency ties CoT on MMLU (96.7%) and GSM8K (76.7%) through majority voting across diverse samples
  • ToT excels on math (76.7%, +10% over Standard) by exploring multiple solution branches before committing
  • ReAct hit the highest ARC-Challenge score (96.0%) through tool-augmented reasoning, grounding answers in external data
The same 4.3B model scores 10 points higher on math just by changing how it thinks, not what it knows.

§

Configure remote Ollama instances via CLI menu, config.yaml, or the Python API (set_ollama_host()). The one-command install script handles everything automatically.

Adding New Agents

Three files to touch. Inherit from BaseAgent, implement stream(query), register in the agent map. The interceptor picks it up automatically.

from agent_reasoning.agents.base import BaseAgent class MyNewAgent(BaseAgent): def stream(self, query): yield "Thinking differently...\n" # Your custom logic here yield "Final Answer" # Register in AGENT_MAP (interceptor.py) # Add to AGENT_DESCRIPTIONS (agent_cli.py) # Optional: create a visualizer
